# Chocolate 
Chocolate - Batch script to automate software installation in Windows
## Easy install many programs
#### > Fast installation
#### > One click (Run As Administrator)
#### > Safe
# How to install?
## Copy the code in chocolate.bat and save in your notepad.
#### > Run Win+R then digit notepad (click enter)
#### > Paste the code
#### > Save the code as chocolate.bat (Select all type of files)
#### > Run the .bat as administrator
#### > That's All!
#### > *DO YOU CAN ADD MANY PROGRAMS IN THE BAT SCRIPT :)
